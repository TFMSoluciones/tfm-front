UgCS COMMANDER
Perpetua

Descripción
solución para desplegar planes de vuelo con varios equipos de manera simultanea, controlados desde una única interfaz.

Precio
€ 4'390.00

Características

  • Planeacion de las diferentes rutas para cada vehículo por separado
  • Conexión directa vía Wi-Fi
  • Telemetría de todas las aeronaves en una única interfaz
  • Modo gráfico para monitorear el ambiente en 3D
  • Notificaciones y alarmas en caso de proximidad de equipos
  • Cada drone puede ser pausado (hold) y vuelve a continuar siempre que sea necesario

Aplicaciones

  • Aseguramiento de perímetros: Pruebas de sistemas anti-drones que protegen aeropuertos, objetos industriales y otras infraestructuras críticas enviando simultáneamente los drones desde diferentes direcciones hacia el objeto protegido.
  • Agricultura: Rocíe simultáneamente los cultivos en paralelo utilizando múltiples drones.
  • Mapeo: Mapeo de grandes áreas usando múltiples drones simultáneamente.

Drones soportados

Controles DJI soportados: Control SDK, Control Smart, Crystal Sky, Control smart M300
*Como la infraestructura DJI viene cerrada, es necesario trabajar con la aplicación gratuita para Android UgCS for DJI.


Requerimientos Mínimos del Sistema

Sistema operativo

  • Windows 7 with SP1 o posterior; Windows 8; Windows 10/64-bit
  • Mac OS X Maverick 10.9 o posterior / 64-bit
  • Linux: Ubuntu 18.04 LTS / 64-bit

CPU

  • Core 2 Duo o Athlon X2 at 2.4 GHz

Memoria

  • RAM recomendada: 4GB o mas

Tarjeta gráfica

  • Tarjeta gráfica con soporte para DirectX 9 (shader model 2.0). Cualquier tarjeta fabricada desde el 2004 debe funcionar

Disco duro

  • 2 GB de espacio libre

Resolución de pantalla mínima

  • 1024 x 768
